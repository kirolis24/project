CSS

. CSS, short for Cascading Style Sheets, is a language used to format and design web pages.
. CSS helps control the look and feel of elements within an HTML page,
such as colors, fonts, alignment, spacing, and even element movement.
. Instead of repeating formatting for each element, CSS allows you to write rules once and apply them to multiple elements,
keeping your code more organized and easier to maintain.
. CSS can also be used to create responsive designs that fit different screens such as phones and tablets.
. CSS is an essential tool for any web developer who wants to create attractive and user-friendly pages.

CSS commands

1. color Specifies the text color. Example: color: red;
2. background-color Changes the background color of the element. Example: background-color: blue;
3. font-size Specifies the font size. Example: font-size: 16px;
4. font-family Specifies the font type. Example: font-family: Arial;
5. text-align Sets the text alignment (right, left, center). Example: text-align: center;
6. margin Specifies the distance between the element and the surrounding elements (outside). Example: margin: 20px;
7. padding Specifies the distance between the element's content and its border (inside). Example: padding: 10px;
8. border Adds a border around the element. Example: border: 1px solid black;
9. width Specifies the element's width. Example: width: 300px;
10. height Specifies the element's height. Example: height: 150px;
11. display Controls the element's display (block, inline, flex, etc.). Example: display: flex;
12. position Specifies the element's positioning (static, relative, absolute, fixed). Example: position: relative;
13. top, right, bottom, left Used with position to move the element in different directions. Example: top: 10px;
14. z-index Controls the element's position above or below other elements. Example: z-index: 2;
15. overflow Specifies what happens if the element's content overflows. Example: overflow: hidden;
16. opacity Controls the element's transparency. Example: opacity: 0.5;
17. transition Adds animation effects when properties change. Example: transition: all 0.3s ease;
18. box-shadow Adds a shadow around the element. Example: box-shadow: 0 4px 8px rgba(0,0,0,0.3);
19. text-decoration Controls the text decoration (such as underlining or not). Example: text-decoration: none;
20. cursor Changes the shape of the mouse cursor when it approaches the element. Example: cursor: pointer;

Different ways of writing CSS

1.Inline CSS: Applied directly inside HTML elements using the style attribute. It's useful for quick adjustments but makes code harder to manage.

2.Internal CSS: Defined within a <style> tag inside the <head> section of an HTML document. This method helps keep styles organized within a single page.

3.External CSS: Stored in a separate .css file and linked to an HTML document. This is the best method for managing styles across multiple pages.
